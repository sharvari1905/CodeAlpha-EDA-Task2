# CodeAlpha Data Analytics Task 2 - EDA

## Objective
Perform Exploratory Data Analysis on Titanic dataset to uncover survival patterns and factors affecting survival rates.

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Google Colab

## Key Insights
1. **Gender Impact**: 74% females survived vs 19% males - shows 'women and children first' protocol
2. **Class Impact**: 1st class 63% survival vs 3rd class 24% - clear socio-economic bias
3. **Age & Fare**: Children and high-fare passengers had better survival odds

## Visualizations
- Survival Count Distribution
- Survival Rate by Gender 
- Survival Rate by Passenger Class
- Correlation Heatmap of All Features

## Conclusion
Survival on Titanic was not random — it depended heavily on gender, class, and age. This EDA reveals clear patterns in disaster survival demographics.
